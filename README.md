A simple Command Line Interface (CLI) task manager written in Rust. 

Usage

    Commands:
        List Tasks: 
            cargo run -- ls
            cargo run -- list
        Add Task: 
            cargo run -- add "Task Name"
            cargo run -- a "Task Name"
        Remove Task:
            cargo run -- remove [Task ID]
            cargo run -- rm [Task ID]
